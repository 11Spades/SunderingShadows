[1mNAME[0m
       first_inventory() - return the first item in an object's inventory


[1mSYNOPSIS[0m
       object first_inventory( mixed ob );


[1mDESCRIPTION[0m
       Return  the first object in the inventory of <ob>, where <ob> is either
       an object or the file name of an object.
