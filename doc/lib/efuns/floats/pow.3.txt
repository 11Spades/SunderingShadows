[1mNAME[0m
       pow() - find an exponent of a float


[1mSYNOPSIS[0m
       float pow( float x, float y );


[1mDESCRIPTION[0m
       pow() returns x to the y power.  If x is 0.0, y must be positive.  If x
       is negative, y must be an integer.


[1mSEE ALSO[0m
       exp(3), log(3), sqrt(3)
