[1mNAME[0m
       in_input() - determines if a player is inputting to an input_to


[1mSYNOPSIS[0m
       int in_input( object default : F_THIS_OBJECT );


[1mDESCRIPTION[0m
       Returns  1  if  the  object  is  currently  inputting to an input_to or
       get_char.


[1mSEE ALSO[0m
       get_char(3), input_to(3)
