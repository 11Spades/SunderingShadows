[1mNAME[0m
       set_living_name() - set a living name for a living object


[1mSYNOPSIS[0m
       void set_living_name( string name );


[1mDESCRIPTION[0m
       Set  a  living  name  on  an object that is living. After this has been
       done, the object can be found with "find_living()".
