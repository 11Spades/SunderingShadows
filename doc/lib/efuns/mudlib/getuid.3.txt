[1mNAME[0m
       getuid() - return the user id (uid) of an object


[1mSYNOPSIS[0m
       string getuid( object ob );


[1mDESCRIPTION[0m
       Returns  the  user id of an object.  The uid of an object is determined
       at object creation by the creator_file() function.
