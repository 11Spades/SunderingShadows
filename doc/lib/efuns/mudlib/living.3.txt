[1mNAME[0m
       living() - detects whether or not a given object is "living"


[1mSYNOPSIS[0m
       int living( object ob );


[1mDESCRIPTION[0m
       Return true if 'ob' is a living object (that is, if "enable_commands()"
       has been called by 'ob').


[1mSEE ALSO[0m
       interactive(3), enable_commands(3)
