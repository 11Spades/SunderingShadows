[1mNAME[0m
       livings() - return an array of all living objects


[1mSYNOPSIS[0m
       object *livings( void );


[1mDESCRIPTION[0m
       Returns  an  array of pointers to all living objects (objects that have
       had enable_commands() called in them).


[1mSEE ALSO[0m
       enable_commands(3), find_living(3), users(3), objects(3)
